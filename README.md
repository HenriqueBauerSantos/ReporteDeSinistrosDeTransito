# Sistema para reporte de sinistros de trânsito - Front-end em React 
Este projeto é o frontend da aplicação InfoTrânsito, um sistema para registro, visualização e gerenciamento de sinistros de trânsito. A interface está sendo desenvolvida em React, integrando-se à API ASP.NET Core, e atualmente inclui telas de login, cadastro de sinistros, gerenciamento de veículos, pessoas envolvidas e fluxo completo de envio dos dados. O desenvolvimento segue em andamento, com foco em melhorar a usabilidade, validar dados e integrar novas funcionalidades conforme a evolução do backend.
